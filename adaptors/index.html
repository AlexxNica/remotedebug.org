Protocol Adaptors

This is an overview of the community-built protocol adaptors for browsers and runtimes.

  • RemoteDebug Firefox Adaptor

    Protocol adaptor for Mozilla Firefox.

  • iOS Webkit Debug Proxy

    Protocol adaptor for Apple Safari (iOS devices).

  • IE Diagnostics Adapter

    Protocol adaptor for Microsoft IE 11+.

  • Node Inspector

    Protocol adaptor for Node.JS.

  • Stetho

    Protocol adaptor for Android apps.

  • Pony Debugger

    Protocol adaptor for iOS apps.
