Tools, services, editors and other integrations that work with RemoteDebug

This is an overview of the community-built clients, tools and services, that's using the Chrome Remote Debugging protocol.

Clients:

  • chrome-remote-interface

    Node client for Chrome Remote Debugging

Gateways:

  • RemoteDebug Gateway

    A gateway for RemoteDebug (Chrome Remote Debugging) connections, that allows you to connect a client to multiple endpoints.

Editors:

  • Lighttable

    Light Table is a new interactive IDE that lets you modify running programs and embed anything.

  • Sublime Web Inspector

    Debug Javascript right in the Sublime Text editor.

  • Adobe Brackets

    Brackets is an open source code editor for web designers and front-end developers.

  • vim-chrome

    Vim meets Chrome. Experiments at driving a remotely running Chrome instance from Vim.

  • Visual Studio Code: Chrome Debugger

    Chrome Debugger for Visual Studio Code

Tools:

  • WebDriver for Chrome

    The Selenium WebDriver implementation for Chrome that users Chrome debugging protocol.

  • Chromium Telemetry

    Telemetry is a performance testing framework used by the Chromium project.

  • crconsole

    Remote JavaScript console for Chrome/Webkit.

Services:

  • Calibre

    Performance benchmarking service who's using remote debugging for Chrome behind the scenes.
