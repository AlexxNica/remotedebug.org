RemoteDebug specifications

  • RemoteDebug mDNS Device Discovery Specifiction

    Specification for annnoncing RemoteDebug compatible devices over mDNS/Bonjour.

Specifications for existing remote debugging protocols

  • Chrome Debugger Protocol

    Used by Google Chrome and Chromium-based browsers like Opera.

  • WebKit Remote Debug Protocol

    Used by Apple Safari and WebKit based browsers

  • Mozilla debugging protocol

    Used by Mozilla Firefox and Mozilla FirefoxOS.

  • Firebug CrossFire

    Available for Firefox via an extension (maybe?)

  • Opera Scope Transport Protocol (deprecated)

    Used by older versions of Opera before they moved to Chromium.
